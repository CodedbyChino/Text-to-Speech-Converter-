Instructions

Step 1: Set Up an AWS Account
If you haven't already, sign up for an AWS account. You can use the free tier for this project.

Step 2: Create an IAM Role
Create an IAM role with permissions to execute Lambda functions and access Amazon Polly. Attach the following policies:

AWSLambdaBasicExecutionRole
AmazonPollyReadOnlyAccess

Step 3: Create a Lambda Function
- Go to the Lambda console.
- Click on "Create function".
- Choose "Author from scratch".
- Name your function (e.g., "TextToSpeechConverter").
- Choose Python as the runtime.
- Choose the IAM role you created earlier.
- Click "Create function".

Step 4: Write Lambda Function Code

Step 5: Deploy Lambda Function
Click on "Deploy" to deploy the Lambda function.

Step 6: Create an API Gateway
- Go to the API Gateway console.
- Click on "Create API".
- Choose "REST API".
- Click "Build".
- Name your API (e.g., "TextToSpeechAPI").
- Click "Create API".

Step 7: Create a Resource and Method
- Click on "Actions" -> "Create Resource".
- Name your resource (e.g., "convert").
- Click on "Create Resource".
- Select the newly created resource, then click on "Actions" -> "Create Method".
- Choose "POST" method.
- Select "Lambda Function" as the integration type.
- Choose the region where your Lambda function is deployed.
- Type the name of your Lambda function.
- Click "Save".

Step 8: Deploy API
- Click on "Actions" -> "Deploy API".
- Create a new stage (e.g., "prod").
- Click "Deploy".
- Note down the invoke URL provided.
